Map Factory Framework

该框架允许您通过 Factory 和可视化抽象（visualization abstractions）创建地图。该系统把 ScriptableObjects 的配置设置序列化，以便于设计师使用。

地图控件（Map Controller）

不是指地图可视化（Map Visualization）结构的一部分，而是指应用的其余部分，将请求从 Map Visualization 获取切片。

Map Visualization

可视化树的核心类（class），用于保存和管理 factories。

Factories

Factory 是 Mapbox Api 终端地址（end points）周围的封装器（wrappers），这样的终端地址包括栅格图像或矢量数据等。

Terrain Factory

该 Factory 负责创建地图的地形底图（base）。

Map Image Factory

该 Factory 负责将素材和纹理分配给地图底图网格（base mesh）。

Mesh Factory

该 Factory 负责矢量数据的可视化。
