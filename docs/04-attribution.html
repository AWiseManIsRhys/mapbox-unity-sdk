归属声明（Attribution）

使用 Mapbox 的自定义地图和数据必须注明 Mapbox 和其他涉及到的数据提供者。Mapbox 的定制设计受版权保护，您须标明我们数据源的归属信息。所有级别套餐的用户都须遵守这一规定。

您必须在使用了 Mapbox Unity SDK 的所有地图上均标注 Mapbox 的文字商标和归属声明。我们给 SDK 提供了包含所有必需信息的Attribution预设组件。该预设组件使用 UGUI 进行整合和定制。您可以调整 Mapbox 文字商标和归属声明的位置（方框转换的枢轴和锚点），但它们在地图上必须保持可见状态。您还可以更改方框变换的背景颜色（默认为透明）和归属声明文字的颜色，以最佳地匹配您的设计审美，但所有信息必须清晰可读。此外，您不得更改 Mapbox 字标或文字归属声明。如果您希望改变 Mapbox 位置或移除 Mapbox 字样，请联系我们的销售团队，讨论在我们的企业套餐中是否有其他适合的方案。
