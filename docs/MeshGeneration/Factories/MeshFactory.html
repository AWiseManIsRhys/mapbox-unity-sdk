Mesh Factory

该 Factory 负责矢量数据的可视化。

图层查看器（Layer Visualizers）

矢量图层查看器（Vector Layer Visualizer）

在 3D 空间中为多边形和直线要素创建可视化

点图层查看器（Poi Layer Visualizer）

在 2D/3D 空间中为点要素创建可视化

Modifier Stack（修饰符堆栈）

Modifier Stack（修饰符堆栈）

修饰符堆栈跟要素的样式差不多。堆栈保持网格和游戏对象修饰符，并在要素对象上运行这些修饰符。这是为要素创建可视化的实际过程。

Merged Modifier Stack（合并的修饰符堆栈）

运行方式和常规堆栈一样，但是会合并输出的网格。

修饰符（Modifiers）

网格修饰符（Mesh Modifiers）

网格修饰符用于创建数据，如顶点，三角形，法向量和 UV，以构建网格。

Chamfer Modifier（斜削角修饰符）

添加到另一个顶点和线条更平滑的斜削角。仅与 Polygon Mesh Modifier 配合使用，用于创建平滑的建筑网格。

Polygon Mesh Modifier（多边形网格修饰符）

利用多边形要素的顶点创建多边形。主要用于创建建筑物占地面积，土地用途，水域等网格。

Line Mesh Modifier（直线网格修饰符）

利用直线要素的顶点创建多边形。主要用于创建道路，边界，等高线（contour line）网格。

Height Modifier（高度修饰符）

根据要素高度值提升多边形高度，并创建侧壁，使其具有最小高度值或零高度（ground）。

UV Modifier（UV 修饰符）

为多边形创建 UV 地图值。与 Polygon Mesh Modifier 配合使用。

Game Object Modifiers（游戏对象修饰符）

游戏对象修饰符使用网格来处理和修改已经存在的游戏对象。

Texture Modifier（纹理修饰符）

改变给定的游戏对象的材质（material）。
