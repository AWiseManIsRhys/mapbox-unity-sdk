Terrain Factory

Terrain Factory 创建地图底图网格。它能够创建平坦的地形，真实的高度地形或修改过的地形（用实际高度乘以一个因数）。

参数（Parameters）

地图类型 ：Flat（平坦），Height（高度）或 Modified Height（修改过的高度）。

地图 ID 类型 ：用于Height 和 Modified Height地图类型的数据源。默认值为“mapbox.terrain-rgb”。

高度乘数（Height Multiplier） ：用于Modified Height地图类型作为查询地形高度的乘数。

分辨率（Resolution） ：地形生成为栅格，分辨率表示此网格每条边上的样本计数。

Material ：生成的网格的底图材料。很有可能被 MapImageFactory 覆盖，但如果没有使用 MapImageFactory，Material 将很有用。

更新 ：更新功能使用当前参数重新创建地形。

方法

Terrain Factory 有三种公用方法；Initialize ， Register 和 Update（来自底图 Factory 类）。

Initialize方法清除并重新初始化已注册的切片列表。可以用作复位方法。

Register方法将新的切片添加到待处理的已注册切片中。

Update方法清除所有切片的网格数据，并使用当前设置重新绘制。首先清除网格数据对于拼接瓦片边缘很重要。

私有方法 Run 启动切片处理，并根据 Map Type 参数给切片添加不同的功能。这些方法将来可以提取到外部模块中。

CreateTerrainHeight从 Mapbox 服务器中获取数据，并将其传递给 GenerateTerrainMesh 进行网格生成，从而创建一个网格，并查询该网格中每个节点的高度数据。网格顶点，三角形，法向量和 UV 地图都在该函数中创建并设置。

FixStitches 方法可以拼接地形切片的边缘，使其连续且平滑。

CreateFlatMesh 创建一个四边形作为底图网格，同时不使用分辨率。
