入门

从 Unity Package 获取安装包

  1. 访问 https://www.mapbox.com/unity/

  2. 单击 Download the SDK

  3. 在 Unity 中导入程序包

    • Assets—>Import package—>Custom Package（从 Unity 编辑器菜单）

      或者

    • 双击已下载的 .unityPackage

从 Git 获取安装包（数据源）

  1. git clone https://github.com/mapbox/mapbox-unity-sdk.git
  2. 了解如何配置项目并学习如何贡献您的资源
  3. 实际的 Unity 项目是在 sdkproject 目录中

配置您的 Mapbox API 令牌

您需要访问令牌（acess token）才能使用您的 Mapbox 帐户功能。

  • 获取 Mapbox API 令牌并将其添加到 Unity 项目中
